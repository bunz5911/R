# Cloudflare Pages 프로젝트 목록으로 이동하기

## 📍 현재 위치

현재 **GitHub 통합 상태 페이지**를 보고 계십니다. 이것은 프로젝트 설정 페이지가 아닙니다.

## ✅ 다음 단계: Pages 프로젝트 목록으로 이동

### Step 1: 왼쪽 사이드바에서 Pages 클릭

화면 왼쪽 사이드바를 확인하세요:
- "Code, planning, and automation" 섹션 아래에 **"Pages"** 메뉴가 보입니다
- **"Pages"**를 클릭하세요

### Step 2: 프로젝트 목록 확인

Pages 메뉴를 클릭하면:
- Pages 프로젝트 목록이 나타납니다
- "rakorean" 프로젝트가 보일 것입니다
- 또는 "Create a project" 버튼이 보일 것입니다

### Step 3: 프로젝트 선택

**옵션 A: "rakorean" 프로젝트가 이미 있는 경우**
1. 프로젝트 목록에서 **"rakorean"** 클릭
2. 프로젝트 페이지로 이동

**옵션 B: 프로젝트가 없는 경우**
1. **"Create a project"** 버튼 클릭
2. Git 저장소 연결: `bunz5911/R`
3. 프로젝트 설정 진행

### Step 4: Settings 탭으로 이동

프로젝트 페이지에서:
1. 상단 탭 메뉴 확인:
   ```
   [Overview] [Deployments] [Settings] [Custom domains] ...
   ```
2. **"Settings"** 탭 클릭

### Step 5: Builds & deployments 섹션 찾기

Settings 페이지에서:
1. 왼쪽 사이드바 또는 페이지 중앙에 섹션 목록이 보입니다
2. **"Builds & deployments"** 섹션 찾기
3. 클릭하여 열기

### Step 6: Build output directory 확인

**"Builds & deployments"** 섹션에서:
- **"Build output directory"** 필드 찾기
- 값이 **`.` (점 하나)** 인지 확인

---

## 🔍 단계별 요약

1. **왼쪽 사이드바** → **"Pages"** 클릭
2. 프로젝트 목록에서 **"rakorean"** 클릭 (또는 새로 생성)
3. 상단 탭에서 **"Settings"** 클릭
4. **"Builds & deployments"** 섹션 찾기
5. **"Build output directory"** 필드 확인

## 💡 팁

### 빠른 접근 방법

왼쪽 사이드바의 **"Pages"** 메뉴를 클릭하면 바로 Pages 프로젝트 목록으로 이동할 수 있습니다.

### 프로젝트가 보이지 않는 경우

- **"Create a project"** 버튼을 클릭하여 새로 만들 수 있습니다
- Git 저장소: `bunz5911/R` 연결
- 프로젝트 이름: `rakorean-pages` (또는 원하는 이름)
