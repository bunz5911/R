# HanaTalk (하나톡) - Korean Learning Community

A premium, glassmorphism-styled web application for foreigners learning Korean.

## Features
- **Sentence Practice (문장 연습):** Practice reading and listening to common Korean sentences.
- **Pronunciation Lab (발음 연습):** Mock interface for recording and checking pronunciation.
- **Community Feeds:**
    - **Grammar (문법):** Ask questions about tricky particles and conjugations.
    - **Tips (꿀팁):** Share and find real-life survival tips for living in Korea.
    - **Culture (문화):** Discuss cultural differences and etiquette.

## Tech Stack
- **HTML5**: Semantic structure.
- **CSS3**: Modern "Glassmorphism" design, CSS Variables, Flexbox/Grid, Animations.
- **JavaScript (Vanilla)**: SPA-like navigation, dynamic content rendering.

## How to Run
Simply open `index.html` in your web browser. No build step required!
