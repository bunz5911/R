# 동화 이미지 가이드

## 📐 이미지 사양

### 크기:
- **너비**: 780px
- **높이**: 440px
- **비율**: 16:9 (약간 더 넓음, 정확히는 780:440 = 1.77:1)
- **포맷**: JPG (권장) 또는 PNG
- **용량**: 100-300KB (최적화 권장)

### 품질:
- **해상도**: 고해상도 (Retina 대응)
- **선명도**: 최대한 선명하게
- **압축**: 약간의 압축 (품질 80-90%)

## 🎨 디자인 가이드

## 📝 텍스트 오버레이

**텍스트는 CSS로 자동 처리됩니다!**

이미지 위에 다음이 자동으로 표시됩니다:
- 번호: "1", "2", "3" 등 (좌측 상단)
- 제목: "강아지 닥스훈트의 비밀" (중앙 하단)
- 배경: 반투명 검은색 그라디언트
- 그림자: 텍스트 가독성 향상

**이미지에는 텍스트를 넣지 마세요!**
- 순수 이미지만 제작
- 텍스트는 CSS로 처리

## 🎨 제작 툴 (추천)

### Figma:
```
1. 780×440 Frame 생성
2. 배경 이미지 삽입
3. Export → JPG (80% 품질)
```

### Photoshop:
```
1. 780×440px 캔버스
2. 디자인 작업
3. Export for Web → JPG (High)
```

### AI 이미지 생성:
```
프롬프트: "cute dachshund puppy, children's book illustration, warm colors, friendly, Korean fairy tale style"
사이즈: 780×440px
스타일: illustration, cartoon, children's book
```
