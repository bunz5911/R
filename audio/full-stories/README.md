# 전체 이야기 오디오 파일

이 폴더에는 50개 동화의 전체 읽기 MP3 파일이 저장됩니다.

## 파일 명명 규칙
- `story-1.mp3` - 강아지 닥스훈트의 비밀
- `story-2.mp3` - 공룡발자국
- `story-3.mp3` - 기린
- ...
- `story-50.mp3` - 희망

## 파일 사양
- 포맷: MP3
- 비트레이트: 128kbps 이상 권장
- 샘플레이트: 44.1kHz
- 채널: Mono 또는 Stereo

## 사용 방법
1. MP3 파일을 `story-{동화ID}.mp3` 형식으로 이름 변경
2. 이 폴더에 복사
3. 앱에서 "전체 이야기 듣기" 버튼 클릭 시 자동 재생

## 주의사항
- 파일명은 반드시 `story-1.mp3` ~ `story-50.mp3` 형식을 따라야 합니다.
- 파일이 없으면 에러 메시지가 표시됩니다.
